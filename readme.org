* unix-cmd

common lispからシェルコマンドを実行する方法はいくつもある。

- trivial-shell
- asdfのuiop
- sbclのsb-posix

探せば他にいくらでも出てくる気はする。 
ただ、それぞれ使い方を新たに覚えるのは面倒だし、 
個人的にはなるべくunixコマンドの原型を保ったまま同じ感覚で使いたいので、
少しずつ気が向いたら作ってみる事にした。
